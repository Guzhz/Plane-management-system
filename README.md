# Plane-management-system
飞机订票管理系统
### 包说明
整个项目包括：源代码、流程图、实训报告、数据库表
    Database - mysql数据库表
    Flowchart - 流程图
    Jar - 依赖jar包
    PlaneSystem - 源代码
    ShowPicture - 展示图

### 程序说明
本飞机订票系统模仿各大售票网站，支持根据日期、始发地、目的地等搜索。
管理员端可修改航班
乘客端可顶、退、改票
